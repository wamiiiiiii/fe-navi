# FE ナビ — 基本情報技術者試験 学習PWA

基本情報技術者試験（FE）の学習を支援するPWAアプリケーション。教科書・問題演習・用語辞書をオフラインで利用できる。

## 公開URL

`https://wamiiiiiii.github.io/fe-navi/`（GitHub Pages 自動デプロイ）

## 試験範囲

| 科目 | 内容 | 試験形式 |
|---|---|---|
| 科目A | テクノロジ系・マネジメント系・ストラテジ系の知識問題 | 60問・90分・4択 |
| 科目B | プログラミング・アルゴリズム・情報セキュリティの応用問題 | 20問・100分・疑似言語コード読解 |

合格基準：科目A 60点以上 かつ 科目B 60点以上。

## ロードマップ

| Phase | 内容 | 状態 |
|---|---|---|
| **0** | iPass-navi をベースにスケルトン構築（リブランド・空データ） | 進行中 |
| **1** | 科目A：4択問題演習・教科書・用語辞書 | 計画中 |
| **2** | 科目B：疑似言語コード表示＋4択（基本UI） | 計画中 |
| **3** | 科目B：変数トレース／ステップ実行UI（本格） | 計画中 |
| **4** | 過去問取り込み（IPA公開分）・類題自動生成 | 計画中 |

## ローカル起動

```bash
# 静的サーバーで配信
python3 -m http.server 4173
# → http://localhost:4173 を開く
```

## E2E テスト

```bash
npm install
npx playwright install chromium webkit
npm run test:e2e
```

## ベースアプリ

このアプリは [iPass ナビ（ITパスポート学習PWA）](https://github.com/wamiiiiiii/ipass-navi) のコードを土台にしている。共通基盤の改修は両アプリ双方に反映する想定。
